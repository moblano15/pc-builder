Documentación del Proyecto PC Builder

1. Introducción

PC Builder es una aplicación web que permite a los usuarios seleccionar componentes de computadora y calcular el precio total de su configuración. La aplicación está desarrollada con tecnologías web estándar (HTML, CSS, JavaScript) sin dependencias externas.

Características principales:
  • Interfaz intuitiva con diseño responsive
  • Base de datos de componentes reales con imágenes
  • Cálculo automático del precio total
  • Visualización de especificaciones técnicas
  • Temática "Matrix" (fondo negro con texto verde)

2. Estructura del Proyecto

pc-builder/
├── index.html # Página principal
├── style.css # Estilos CSS
├── script.js # Lógica de la aplicación
├── data.js # Base de datos de componentes
├── contenido.html # Documentación adicional
└── instalar_pc_builder.sh # Script de instalación

2.1. index.html

Estructura básica de la aplicación con:

  • Formulario para selección de componentes
  • Tabla de resumen con detalles y precios
  • Sección de totalización

2.2. style.css

Estilos con temática "Matrix" que incluyen:

  • Diseño responsive para diferentes tamaños de pantalla
  • Efectos visuales para los componentes seleccionados
  • Estilización de tablas y formularios

2.3. script.js

Lógica principal que maneja:

  • Generación dinámica del formulario
  • Actualización de la tabla de resumen
  • Cálculo del precio total

2.4. data.js

Base de datos con información real de componentes:

  • 8 categorías diferentes (CPU, GPU, RAM, etc.)
  • 2-3 opciones por categoría
  • Imágenes, especificaciones y precios actualizados

3. Funcionamiento de la Aplicación

3.1. Diagrama de Flujo

  1. El usuario carga la aplicación
  2. Se genera dinámicamente el formulario de selección
  3. El usuario selecciona componentes
  4. La tabla de resumen se actualiza automáticamente
  5. Se calcula y muestra el precio total

3.2. Funciones Principales

Función Descripción Archivo
renderForm() Crea dinámicamente los controles del formulario basados en data.js script.js
renderTable() Actualiza la tabla de resumen con los componentes seleccionados script.js
selected Objeto que almacena los componentes seleccionados por el usuario script.js

4. Base de Datos de Componentes

La aplicación incluye 8 categorías de componentes con especificaciones técnicas reales:

CPU (Procesadores)

  • Intel i5-10400F (6 núcleos, 12 hilos)
  • Ryzen 5 5600G (6 núcleos, 12 hilos)

GPU (Tarjetas Gráficas)

  • NVIDIA RTX 3060 (12GB GDDR6)
  • NVIDIA RTX 4060 (16GB GDDR6)

RAM (Memoria)

  • Corsair Vengeance RGB Pro 32GB DDR4
  • Corsair 32GB DDR5 5200MHz

Y además: Placas base, refrigeración, almacenamiento, fuentes de alimentación y cajas.

5. Instalación y Uso

5.1. Requisitos

  • Sistema operativo Linux (recomendado Ubuntu 22.04/24.04)
  • Python 3 para el servidor de desarrollo
  • Navegador web moderno (Chrome, Firefox, Edge)

5.2. Pasos de Instalación

# 1. Descargar el script
wget https://ejemplo.com/instalar_pc_builder.sh

# 2. Dar permisos de ejecución
chmod +x instalar_pc_builder.sh

# 3. Ejecutar el script
./instalar_pc_builder.sh

# La aplicación se abrirá automáticamente en http://localhost:8000

5.3. Script de Instalación

El script instalar_pc_builder.sh realiza las siguientes acciones:

  1. Crea la estructura de directorios
  2. Genera todos los archivos necesarios
  3. Inicia un servidor web local
  4. Abre la aplicación en el navegador predeterminado

6. Personalización y Extensión

6.1. Añadir Nuevos Componentes

Para agregar nuevos componentes, edita el archivo data.js:

// Ejemplo: Añadir nueva CPU
{
  name: "Intel i7-12700K",
  socket: "LGA1700",
  tdpMin: 125,
  tdpMax: 190,
  image: "url_de_la_imagen.jpg",
  specs: "12 núcleos (8P+4E), 20 hilos, hasta 5.0GHz",
  price: 420
}

6.2. Modificar el Diseño

Edita style.css para cambiar la apariencia:

/* Cambiar tema de colores */
body {
  background: linear-gradient(135deg, #000033, #000066);
  color: #00ccff;
}

/* Modificar estilos de la tabla */
table {
  border: 2px solid #00ccff;
}

6.3. Añadir Nuevas Funcionalidades

Puedes extender script.js con nuevas características:

  • Compatibilidad entre componentes
  • Guardar configuraciones
  • Comparar componentes

7. Conclusión

PC Builder es una aplicación completa para ensamblar computadoras virtuales con:

  • Interfaz intuitiva y atractiva
  • Datos técnicos reales
  • Fácil personalización
  • Sin dependencias externas

Ideal para entusiastas de hardware, tiendas de computación o como proyecto educativo para aprender desarrollo web.
